List recent long evaluation reports. Execute:

```bash
muscle long-eval reports
```

Shows a table of recent deep evaluation reports with issue counts and dates.

To run a new deep evaluation:

```bash
muscle long-eval run
```

To clean up old reports:

```bash
muscle long-eval cleanup --force
```

To run a benchmark comparison between review workflows:

```bash
muscle long-eval benchmark
```
